# getMatchedCSSRules
getMatchedCSSRules for Firefox, IE and Chrome

### Installation

for Bower

    bower install getMatchedCSSRules
    
for Browser

    <script type="text/javascript" src="path-to-getMatchedCSSRules"></script>

### Usage
    
    var rules = getMatchedCSSRules(document.body);
    console.log(rules);
